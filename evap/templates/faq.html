    How should I configure the evaluation period of my course?

    There are different criteria for the period:

    • It should be at least one week long.
    • It should be after the last lecture.
    • It should be before the written exam or the first oral exam.
    Of course there are courses for which it is not possible to fulfil all criteria. Please contact the student representatives if you are not sure.
